# View all scalars specifier

A pair to use in the `data` parameter of `viewer` to expose all base
scalars with their original names. See the Julia documentation for
details.

## Usage

``` r
VIEW_ALL_SCALARS
```

## Format

A named list representing the pair ALL_SCALARS =\> "="

## Value

A named list specifier to expose all scalars.
