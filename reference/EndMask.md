# EndMask query operation

Finish specifying a mask to apply to an axis, following BeginMask or
BeginNegatedMask. See the Julia
[documentation](https://tanaylab.github.io/DataAxesFormats.jl/v0.2.0/queries.html#DataAxesFormats.Queries.EndMask)
for details.

## Usage

``` r
EndMask(...)
```

## Arguments

- ...:

  Additional arguments needed to support usage of pipe operator

## Value

A query operation object
